# Contributing

## Local development
1. Clone repository.
2. Open `index.html` in your browser.
3. Use Demo mode for UI-only verification.

## Coding conventions
- Keep the app single-file unless a refactor is intentional.
- Reuse existing naming patterns (`S`, `pStatus`, `render`).
- Prefer small, incremental commits.

## PR checklist
- [ ] UI renders in signed-in and demo modes
- [ ] Search and status filtering work
- [ ] CSV export still works
- [ ] PDF export opens print dialog and includes filtered results
